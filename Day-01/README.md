# Day 1 - Modern JavaScript (ES6)

## 📌 Objective
The objective of Day 1 is to learn the fundamental ES6 features that are widely used in modern JavaScript development. These features help write cleaner, shorter, and more efficient code.

---

## 🚀 Topics Covered

### 1. let and const
- `let` is used for variables whose values can change.
- `const` is used for variables whose values remain constant.

### 2. Arrow Functions
- A shorter syntax for writing functions.
- Makes code more readable and concise.

### 3. Template Literals
- Used to embed variables directly inside strings using backticks (` `).

### 4. Object Destructuring
- Allows extracting values from objects into variables easily.

### 5. Spread Operator (...)
- Used to copy or combine arrays and objects.

### 6. Rest Operator (...)
- Allows a function to accept multiple arguments as an array.

### 7. Default Parameters
- Provides default values to function parameters.
